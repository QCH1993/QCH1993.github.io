Escape Plan, About Memory(2017-09-23,on the way musical festival)

Posted on 2017-09-24 | 0 comments | Visitors

音乐对于我来说，魅力在于它和记忆紧紧相连。

当《时代之梦》的旋律响起，在炫目的灯光和磅礴的大雨浸染之下，所有记忆都能在始料不及的一瞬间闪回到那个特定的时间点。

那个阴暗潮湿充满烟味的沙发，那个冲了一百又一百的柜台，那个不眠不休以期在另一个世界里能逃避窒息的年轻人。那段时间对于我来说，就是逃跑的每一首歌。

逃跑的歌不多，歌单里二十多首，每一首都单曲循环过，至今“一万次悲伤”还以451次的记录牢牢控制着我云村的榜首位置。每次骑着破车去那个“八块钱一小时”的地方挥霍的路上，耳机里总是会响起他们的旋律。

去年大约这个时候腾讯出了一个节目叫“大事发声”，第七期请了逃跑做了live，很赞。赞到我每次去开黑之前都先打开这段视频听一段净化心灵再愉快掉分。可能就是这样一个习惯让2016年整个下半年，在我的记忆中都紧紧的与这个乐队扣在一起。

“我早已忘记一封情书该怎么开头，却忽然想起你是怎么松开我的手”。我大毛川的声音还是一如既往干净的勾人魂魄，以至于后来回看自己拍的视频，听到自己的鬼哭狼嚎都想扇自己几个巴掌（你特么就不能安安静静的当一个美男子？！）。

年轻就是要去燥就是要去爱啊，但是余震可能远远比我自己想象的要强。从心理上自己给自己一个评判，说实话，我觉得自己其实一直并没有做好重新再去了解别人的准备，也没有做好我家大门常打开开怀容纳天地的准备。然而渴望被别人关心的感觉却像是明日香的童年一样，一直折磨着自己。

雨越下越大，把镜片也模糊，有一种梦幻的错觉

“每一颗眼泪 是一万道光 最昏暗的地方也变的明亮”。所谓痛苦的都是闪亮的，必将在之后的人生里提供前行的路灯。每一滴痛苦凝练成的眼泪都能幻化为光，指引前方。那些无眠的夜，那些挣扎的心，终将在多年以后彰显出它的价值。

“凝视我双眼你敢不敢 并不需要语言 心跳能让你看见 你却视而不见”。不敢。有些话不能说出口的就不要说出口。不要相信那些再不疯狂就老了的鬼话，不要被那些热血澎湃的鸡血怂恿，考虑好后果，所有的一切都需要我亲自承担。多问问值不值，少问问悔不悔。

大雨倾盆，愿你我年少的心永不熄灭。
